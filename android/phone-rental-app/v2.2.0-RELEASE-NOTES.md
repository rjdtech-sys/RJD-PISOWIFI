# Phone Rental App v2.2.0 - Default Launcher Update

## Release Summary

**Version:** 2.2.0 (Build 17)  
**Release Date:** April 25, 2026  
**APK File:** `AJC-Phone-Rental-v2.2.0-DefaultLauncher.apk`  
**File Size:** 6.9 MB  

---

## What's New

### 🔒 Improved Kiosk Mode Security (No Device Owner Required!)

This update solves the problem where users could escape kiosk mode by opening other apps and pressing the HOME button.

**Key Features:**
- ✅ **HOME Button Trap** - App now prompts to be set as default launcher
- ✅ **Default Launcher Dialog** - Clear instructions appear when app starts
- ✅ **Better Fallback Mode** - Works on devices where Device Owner cannot be set
- ✅ **Enhanced Instructions** - Step-by-step guide for users

---

## How It Works

### When the App Starts (Without Device Owner):

1. **Dialog appears:** "Set as Default Launcher"
2. **User reads instructions:** Explains why it's needed for kiosk mode
3. **User clicks "Set Now":** Opens launcher selection screen
4. **User selects:** Choose "Phone Rental" app
5. **User selects "Always":** Makes it permanent

### After Setting as Default Launcher:

- ✅ **HOME button** always returns to your rental app
- ✅ **Users can open whitelisted apps** from the app grid
- ✅ **Status bar blocked** (overlay service still active)
- ✅ **Better kiosk security** even without Device Owner

## Installation Options

### Option 1: Via Admin Panel (Recommended)

1. Open admin panel in browser
2. Go to **Phone Rental** → **APK Installer** tab
3. Connect your device via USB
4. Click **"Install APK"** button
5. Wait for installation to complete

### Option 2: Via ADB Command

```bash
adb install -r /opt/ajc-pisowifi/android/phone-rental-app/AJC-Phone-Rental-v2.2.0-DefaultLauncher.apk
```

1. Open the Phone Rental app on the device
2. The app will automatically detect the update (v2.2.0)
3. Tap **"Update Now"** when prompted
4. Wait for download and auto-installation
5. App will restart automatically

## Important Notes

### ⚠️ First Launch After Update:

When the app starts for the first time after update:

1. You'll see a **dialog asking to set as default launcher**
2. **DO NOT SKIP THIS** - it's critical for kiosk security
3. Click **"Set Now"**
4. Select **"Phone Rental"** app from the list
5. Choose **"Always"** (not "Just once")

### 📋 What This Update DOES:

- ✅ Traps HOME button (returns to rental app)

## Troubleshooting

### Problem: Dialog doesn't appear
**Solution:** 
- App may already be default launcher
- Check: Settings → Apps → Default apps → Home app
- Should show "Phone Rental"

### Problem: User can still use Recent Apps button
**Solution:**
- This requires Device Owner mode
- Factory reset the device and set Device Owner during setup wizard
- OR: Disable Recent Apps button in Developer Options

### Problem: App not in launcher selection list
**Solution:**
- Reinstall the APK
- Enable the launcher alias in app settings
- Restart the device
